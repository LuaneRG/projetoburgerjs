# projetoburgerjs

## Descrição:

Este é um estudo de caso de um sistema de compras online de hambúrgueres desenvolvido com JavaScript. O projeto utiliza um JSON Local para a criação dinâmica dos elementos HTML. Foram utilizadas tecnologias como `HTML5` para estruturação semântica, `CSS3` para responsividade e `JavaScript (ES6+)` para manipulação de DOM e lógica do carrinho de compras.

## Como usar:

Este cardápio digital focado em front-end permite a simulação de compra de um produto. Selecione o(s) hambúrguer(es), escolha o tamanho da batata frita (ou a opção sem acompanhamento), adicione ao carrinho e veja o valor final com cálculo automático.
